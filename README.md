# ShopMe – E-Commerce Web Application
ShopMe is a full-stack e-commerce web application built using Django (Backend) and HTML, CSS, JavaScript (Frontend). The platform provides a smooth and secure online shopping experience where users can browse products, manage carts, and place orders efficiently.
# Features
# User Features
User Registration & Login Authentication
Browse Products with Category Filtering
Product Detail View
Add to Cart / Remove from Cart
Secure Checkout System
Order Placement & Order History
# Admin Features
Admin Dashboard (Django Admin Panel)
Add / Update / Delete Products
Manage Categories
Manage Orders
User Management
# Tech Stack
